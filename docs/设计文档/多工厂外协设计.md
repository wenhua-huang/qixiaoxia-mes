# 多工厂外协设计

## 多工厂/租户隔离（factory_id）

**原则**：`factory_id` = 这行数据属于哪个工厂。**所有表都加，零例外。**

**`qxx_md_factory`**：工厂定义表（mes-md.sql 表1）。`factory_id` 自增，`factory_code` 唯一（如 SX-圣享、WL-万隆）。

**DDL 模板**：每张表加同两行：
```sql
factory_id  bigint(20)  not null  comment '工厂ID(关联qxx_md_factory)',
key idx_factory_id (factory_id)
```

**索引策略**：先用单列 `idx_factory_id`，上线后根据慢查询日志按需加联合索引。

**供应商**：归工厂管。同一供应商供多个工厂 → 各建一条。`qxx_md_vendor.factory_id` not null。

**用户-工厂**：`sys_user.factory_id` not null，用户直接归属工厂，不通过部门推导。

**查询约定**：后端 MyBatis 拦截器自动注入 `WHERE factory_id = ?`。跨工厂查询时显式跳过。

**工厂初始化**（平台开发人员直接SQL，见 `qxx_md_factory` 表注释）：
1. 建工厂 → `INSERT qxx_md_factory`
2. 建管理员 → `INSERT sys_user`（带 `factory_id`）
3. 部门/角色/员工由管理员登录后自己建

## 外协（outsource_factory_id）

**问题**：A工厂发包给B工厂。工单上写的是 A工厂建的 vendor_id=5，B工厂员工登录后不知道自己对应 vendor_id=5。

**设计**：`outsource_factory_id` 直接把 vendor 关联到系统工厂。**所有有 `vendor_id` 的外协表统一冗余此字段。**

```
qxx_md_vendor:
  vendor_id=5, vendor_name=万隆纸品, factory_id=1(A), outsource_factory_id=2(B)
       ↑ A工厂建的供应商记录                          ↑ 指向B工厂

qxx_pro_workorder:       factory_id=1, vendor_id=5, outsource_factory_id=2
qxx_pro_task:            factory_id=1, vendor_id=5, outsource_factory_id=2
qxx_pro_route_process:   factory_id=1, vendor_id=5, outsource_factory_id=2
qxx_pro_card_process:    factory_id=1, vendor_id=5, outsource_factory_id=2
qxx_pro_feedback:        factory_id=1, vendor_id=5, outsource_factory_id=2
qxx_wm_outsource_issue:  factory_id=1, vendor_id=5, outsource_factory_id=2
qxx_wm_outsource_recpt:  factory_id=1, vendor_id=5, outsource_factory_id=2
```

**覆盖**：共8张表。`mes-md.sql`（vendor）、`mes-pro.sql`（workorder/task/route_process/card_process/feedback）、`mes-wm.sql`（outsource_issue/outsource_recpt）。

**DDL 模板**：
```sql
outsource_factory_id  bigint(20) default null  comment '外协场景：该供应商对应的系统工厂ID(关联qxx_md_factory)',
```

**为什么冗余**：① 查询一步到位 `WHERE outsource_factory_id = 2`，不 JOIN 不子查询；② 历史追溯不丢——vendor 关系变更后工单仍保留"当时发给谁"；③ 外协关系越频繁变动越需要冗余。

**B工厂员工登录**：
```
手机号 → sys_user → factory_id=2

  ┌─ 我的工厂   WHERE factory_id = 2
  └─ 外协任务   WHERE outsource_factory_id = 2
```

**数据可见性**：

| 视角 | 查询 |
|------|------|
| 自己的数据 | `WHERE factory_id = ?` |
| 外协任务（接单方） | `WHERE outsource_factory_id = ?` |
| 外协工单（发包方） | `WHERE factory_id = ? AND vendor_id = ?` |

**外协供应商创建**（本工厂管理员在UI操作）：
新增外协供应商 → 下拉选系统工厂（如万隆）→ 系统自动写入 `outsource_factory_id` → INSERT `qxx_md_vendor`。

**实施记录**：2026-06-08 全部 124 张表 + `sys_user`/`sys_role`/`sys_dept`/`sys_role_dept`/`sys_user_post`/`sys_role_menu`/`sys_user_role` 已完成 `factory_id` 字段和索引。`qxx_md_vendor_factory`/`qxx_md_client_factory` 已删除。
